# Logo Usage

The Vedik Tattva logo system consists of two wordmarks and an illustration mark. Together these assets form the complete visual signature of the brand.

Consistent usage of these elements ensures recognition, clarity, and reinforces the premium positioning of Vedik Tattva.

Always use the approved logo files. Do not recreate, redraw, or modify the logo.

---

## Logo System

The Vedik Tattva identity is built from three primary assets.

| Asset | Description | Primary Use |
|---|---|---|
| Wordmark A | Primary Vedik Tattva wordmark | Default brand signature |
| Wordmark B | Secondary wordmark variation | Packaging or layout-specific usage |
| Illustration Mark | SVG illustration symbol | Icons, seals, and compact spaces |

These assets are designed to work independently or together depending on context.

---

## Wordmark A

Wordmark A is the **primary brand signature**. It should be used in most brand communications.

Typical use cases:

- Website header
- Primary packaging
- Product labels
- Marketing materials
- Brand presentations

Use this version whenever the full brand name needs to be clearly communicated.

---

## Wordmark B

Wordmark B is a secondary wordmark designed for layouts where the primary wordmark may not fit optimally.

Typical use cases:

- Alternate packaging layouts
- Editorial compositions
- Special marketing formats

Wordmark B should not replace Wordmark A in core brand surfaces unless layout constraints require it.

---

## Illustration Mark

The illustration mark is the symbolic representation of Vedik Tattva. It functions as a compact visual identifier when the wordmark cannot be used.

Typical use cases:

- Favicon
- App icon
- Social media avatars
- Product seals
- Watermarks
- Small digital contexts

The illustration mark may also appear alongside the wordmark in certain brand compositions.

---

## Illustration Variants

Because the illustration mark appears in different contexts, simplified versions are provided.

| Variant | Use Case |
|---|---|
| Full Illustration | Default brand usage |
| Simplified Mark | Small digital contexts |
| Favicon Version | Browser tabs and small UI elements |

Do not redraw or manually simplify the illustration. Only use the approved exported assets.

---

## Background Usage

The logo must always appear on backgrounds that provide clear contrast.

| Background | Recommended Logo |
|---|---|
| Light surfaces | Default logo version |
| Dark surfaces | White or reversed logo |
| Photography | Version with highest contrast |
| Brand color fields | White or reversed logo |

Avoid placing the logo on visually busy backgrounds.

---

## File Formats

Use the appropriate file format depending on the output medium.

| Format | Use Case |
|---|---|
| `.svg` | Websites, apps, scalable digital usage |
| `.png` | Presentations and documents |
| `.pdf` | High‑resolution printing |
| `.eps` | Legacy print workflows |

:::tip Prefer SVG whenever possible
SVG files scale without quality loss and should be used in all digital products.
:::

---

## Asset Structure

Official logo assets are organized as follows.

```
/assets/logos/

  wordmark-a/
    vedik-tattva-wordmark-a.svg
    vedik-tattva-wordmark-a-white.svg

  wordmark-b/
    vedik-tattva-wordmark-b.svg
    vedik-tattva-wordmark-b-white.svg

  illustration/
    illustration-full.svg
    illustration-simplified.svg
    favicon.svg
```

Always download logo files from the official asset library. Avoid exporting logos from screenshots or third‑party documents.

| Variant | Description | Use Case |
|---|---|---|
| Primary | Mark + Wordmark lockup | Default usage |
| Mark Only | Symbol without wordmark | Small sizes, favicons |
| Wordmark Only | Text only | Packaging or editorial layouts |
| Reversed | White logo version | Dark backgrounds |
| Monochrome | Single color | Print limitations |

Do not create new variants outside this set.

---

## Background Usage

The logo must always appear on backgrounds that provide sufficient contrast.

Approved background contexts:

| Background | Logo Version |
|---|---|
| Light surfaces | Primary logo |
| Dark surfaces | White logo |
| Photography | White or dark version depending on contrast |
| Brand color fields | White logo |

Avoid placing the logo on busy or low-contrast imagery.
