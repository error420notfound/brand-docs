# Logo Usage

The Vedik Tattva logo is the most recognizable asset of the brand.  
It represents purity, authenticity, and authority within a category that is often commoditized.

Consistent usage of the logo is critical to maintain recognition and reinforce the premium positioning of the brand.

Always use the approved logo files. Do not recreate, redraw, or modify the logo.

---

## Primary Logo

The **primary logo** consists of the Vedik Tattva wordmark paired with the brand mark.

This version should be used in most brand communications.

Typical use cases:

- Website header
- Packaging front panel
- Product labels
- Marketing materials
- Brand presentations

## Logo Variants

A limited set of logo variants are approved for specific contexts.

| Variant | Description | Use Case |
|---|---|---|
| Primary | Mark + Wordmark lockup | Default usage |
| Mark Only | Symbol without wordmark | Small sizes, favicons |
| Wordmark Only | Text only | Packaging or editorial layouts |
| Reversed | White logo version | Dark backgrounds |
| Monochrome | Single color | Print limitations |

Do not create new variants outside this set.

---

## Background Usage

The logo must always appear on backgrounds that provide sufficient contrast.

Approved background contexts:

| Background | Logo Version |
|---|---|
| Light surfaces | Primary logo |
| Dark surfaces | White logo |
| Photography | White or dark version depending on contrast |
| Brand color fields | White logo |

Avoid placing the logo on busy or low-contrast imagery.

---

## File Formats

Use the appropriate format depending on the output medium.

| Format | Recommended Use |
|---|---|
| `.svg` | Websites, apps, scalable digital use |
| `.png` | Presentations, documents, email |
| `.pdf` | Professional printing |
| `.eps` | Legacy print workflows |

:::tip Prefer SVG whenever possible
SVG files scale infinitely without quality loss and should be used in all digital products.
:::
