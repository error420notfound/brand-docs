# Clear Space & Sizing

## Minimum Clear Space

Always maintain a minimum clear space around the logo equal to the **height of the letter "B"** in the wordmark (we call this unit `1x`).

```
        ┌─────── 1x ────────┐
  1x    │                   │    1x
        │    [  LOGO  ]     │
  1x    │                   │    1x
        └───────────────────┘
```

This ensures the logo is never crowded by other elements.

## Minimum Size

| Usage | Minimum Width |
|---|---|
| Digital (screen) | 80px |
| Print | 20mm |
| Favicon / App Icon | Use mark only |

:::danger Never scale below minimums
Below these sizes, the logo loses legibility. Use the mark-only variant for small contexts.
:::

## Responsive Behavior

On small screens or tight layouts, use the **mark only** instead of the full lockup.

```
Full lockup  →  [●] Brand Name
Mark only    →  [●]
```
