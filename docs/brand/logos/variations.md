# Approved Logo Variations

We provide a defined set of logo variations. Do not create custom variations outside this set.

## Variations at a Glance

| Variant | When to Use |
|---|---|
| **Primary** (mark + wordmark) | Default for most use cases |
| **Mark only** | Small spaces, app icons, favicons, watermarks |
| **Wordmark only** | When paired with the mark separately |
| **Stacked** | Square or tall formats |
| **Reversed / White** | On dark or image backgrounds |
| **Monochrome** | Single-color print, embroidery |

## Color Variants

The logo is approved in:

- **Full color** — on white or very light backgrounds
- **White** — on brand dark, photography, or dark surfaces
- **Black** — on white; use only when color isn't available
- **Single color brand blue** — limited use cases only

:::warning No color modifications
Do not recolor the logo to any color outside the approved set above.
:::
