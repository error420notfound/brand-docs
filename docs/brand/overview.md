# Brand Overview

This section is the **single source of truth** for all brand assets. Everything here is maintained by the Brand team and should be referenced before any external use of our visual identity.

## What's Inside

| Section | Description |
|---|---|
| [Logos](/docs/brand/logos/usage) | Master logo files, clear space rules, approved variations |
| [Colors](/docs/brand/colors/palette) | Primary and secondary palette with hex, RGB, CMYK, and Pantone values |
| [Typography](/docs/brand/typography/typefaces) | Approved typefaces, scale, and usage guidelines |
| [Illustrations](/docs/brand/illustrations/overview) | SVG illustration library and style guide |

## Principles

Our brand identity is built on three pillars:

**Clarity** — Every design decision should make communication clearer, not more complex.

**Consistency** — We look and sound the same everywhere. Teams should use approved assets; don't create alternatives.

**Character** — Within the system, there is room for expression. Guidelines enable creativity — they don't eliminate it.

## Asset Downloads

:::info Download Assets
All production-ready assets (SVGs, PNGs, PDFs) are stored in the `/assets` folder of this repository. Clone the repo to get offline access to all files.
:::
