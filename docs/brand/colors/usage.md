# Color Usage

## The 60-30-10 Rule

A healthy brand layout uses:

- **60%** — Neutrals (Paper, Gray 50, white)
- **30%** — Ink / dark neutrals
- **10%** — Brand Blue or accent color

Overusing accent colors dilutes their impact. Reserve them for CTAs, highlights, and key moments.

## Background / Foreground Pairings

| Background | Foreground | Use Case |
|---|---|---|
| Paper `#FAFAF8` | Ink `#0D0D0D` | Default page background |
| Ink `#0D0D0D` | White `#ffffff` | Dark hero, footer |
| Brand Blue `#1A56DB` | White `#ffffff` | Primary buttons, badges |
| Gray 50 `#F4F4F0` | Ink `#0D0D0D` | Cards, surfaces, sidebars |

## Digital vs Print

| Context | Color Mode |
|---|---|
| Web / App | Hex / RGB |
| Digital advertising | Hex / RGB |
| Offset print | CMYK |
| Premium print | Pantone spot |
| Screen print / embroidery | Pantone spot |

:::caution Print color matching
CMYK conversions are approximations. Always request a proof before a final print run and compare against the Pantone swatch.
:::
